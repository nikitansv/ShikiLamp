# Shikimori for Lampa (Local)

Локальный плагин для Lampa, который добавляет раздел Shikimori: поиск, популярное, онгоинги, анонсы, ручное сопоставление с TMDB и диагностику.

## Версия

0.1.0 — MVP без облачного backend и без полноценной OAuth-авторизации Shikimori.

## Установка в Lampa

1. Разместите файл `dist/plugin.js` на локальном HTTP/HTTPS-сервере, например:
   - `http://192.168.1.10/lampa/shikimori/plugin.js`
   - `https://192.168.1.10/lampa/shikimori/plugin.js`
2. В Lampa откройте **Настройки → Расширения → Добавить плагин**.
3. Введите URL к `plugin.js`.
4. Перезапустите Lampa.
5. В главном меню появится пункт **Shikimori**.

## Ограничения

- Работает в режиме публичного каталога Shikimori (без авторизации пользователя).
- OAuth и синхронизация просмотренных эпизодов не реализованы.
- Прямой доступ к `shikimori.io` из браузерной Lampa возможен при разрешённом CORS, но `User-Agent` и `mixed content` могут блокировать запросы.
- Если Lampa открыта по HTTPS, а плагин/данные загружаются по HTTP, браузер заблокирует запрос (mixed content).
- TMDB-поиск использует внутренний API Lampa (`Lampa.Api.search`). Если он недоступен, автоматический mapping не работает, но остаётся ручной ввод TMDB ID.

## Настройки

Раздел **Shikimori Local** в настройках Lampa:

- Включить плагин
- API Base URL (по умолчанию `https://shikimori.io`)
- Язык результатов
- Показывать кнопку в меню
- Число результатов на страницу
- Порог автоматического mapping
- Автоматически открывать точное соответствие
- Диагностические сообщения
- Очистка кэша, очистка mapping
- Экспорт/импорт mapping
- Экспериментальный access token (отключён по умолчанию)

## Разработка

```bash
cd lampa-shikimori-local
npm install
npm test
npm run build
node --check dist/plugin.js
```

## Структура

```text
src/
  index.js          — точка входа плагина
  config.js         — константы и ключи хранилища
  settings.js       — настройки Lampa
  logger.js         — логирование без утечки токенов
  cache.js          — локальный TTL-кэш
  api/              — клиент Shikimori GraphQL/REST
  mapping/          — хранилище и скоринг mapping
  components/       — экраны Lampa
  ui/               — меню, карточки, шаблоны, стили
tests/              — unit-тесты
```

## Безопасность

- В коде нет `client_secret`, постоянных токенов, API-ключей.
- Экспериментальный токен хранится только локально, не выводится в лог, не передаётся в URL, не попадает в экспорт.
- Плагин не выполняет `eval`, не загружает внешний исполняемый JS, не собирает телеметрию.
