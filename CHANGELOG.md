# Changelog

## 0.1.0 — MVP

### Добавлено
- Раздел Shikimori в главном меню Lampa.
- Главный экран с разделами: Популярное, Онгоинги, Недавно вышедшее, Анонсы, Поиск, Локальные соответствия, Диагностика.
- Поиск аниме по русскому, английскому, оригинальному названию и Shikimori ID.
- Карточка аниме с описанием, типом, статусом, рейтингом, жанрами, студиями.
- Локальное хранилище соответствий Shikimori ↔ TMDB.
- Автоматический поиск соответствия через `Lampa.Api.search` со скорингом.
- Ручной выбор соответствия TMDB с указанием сезона и смещения эпизодов.
- Открытие стандартной карточки Lampa (`Lampa.Router.call('full', ...)`).
- Настройки плагина в Lampa SettingsApi.
- Локальный TTL-кэш для API-запросов.
- Диагностический экран с тестами API, GraphQL, поиска и экспортом отчёта.
- Экспериментальный access token (выключен по умолчанию, скрыт, не логируется).
- Unit-тесты для normalizer, cache, scoring, matcher.

### Известные ограничения
- Нет OAuth и синхронизации прогресса просмотра.
- Автоматический mapping зависит от доступности `Lampa.Api.search`.
- Прямой доступ к `shikimori.io` из браузера может быть ограничен CORS / User-Agent / mixed content.
- Плагин рассчитан на локальную сеть; без LAN-proxy может не работать в некоторых WebView.

### Будущие версии
- OAuth-авторизация Shikimori через пользовательский backend или локальный proxy.
- Синхронизация просмотренных эпизодов с Shikimori.
- Улучшенный matching с учётом MAL ID и TMDB find-by-external-id.
- Поддержка списков пользователя (planned, watching, completed).
- Интеграция с локальным LAN-proxy для обхода CORS при необходимости.
